What is Formatting and why do you do it

When writing code, it's critical that a human understands the program just as well as a computer. To guarantee that everyone understands each other, the code you write must be correctly formatted and indented. Comments can assist maintain consistency, organizing, and making the code more understandable. The markup is simpler to read when the content of elements is indented. The appearance of the page is unaffected by indentation, but it has a significant impact on the readability of the code. Code Formatting Guidelines You normally indent components in HTML markup to highlight that they are nested within each other. There is no set number of times you must indent, but you must ensure that you do so.



The steps for Formatting code
