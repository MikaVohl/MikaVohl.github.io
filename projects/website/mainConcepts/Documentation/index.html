What is Documentation

Within the source code, documentation is written text that explains how the code works. Documentation is important because it allows other programmers to understand their code, which can aid in bug fixes and code maintenance. Writing documentation also makes knowledge transfer to other programmers easier.



How you do documentation

In order to make a comment in HTML, you will have to use < !-- comment --> and for javascript use // if you plan on writing one line documentation or /* */ for a multi line documentation.
Additionally, when starting a new document, make sure you have these basic 4 comments at the top of your source code.
- Name
- Date
- Subject/title
- Brief explanation on the website
